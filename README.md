# Mounts of Mayhem Client (1.21.11)

A lightweight Fabric client mod providing essential HUD utilities without the bloat.

## Features
*   **FPS Counter**: Real-time framerate tracking.
*   **Coordinates**: XYZ and Direction display.
*   **Minimap**: Efficient terrain scanning (Surface heightmap) with Entity dots.
*   **Waypoints**: Press `M` to save locations.
*   **HUD Editor**: Press `Right Shift` to drag and move elements.
*   **Auto-Save**: All configurations save to `.minecraft/config/mayhemclient.json`.

## Controls
*   **M**: Open Waypoint Manager / Add Waypoint.
*   **Right Shift**: Open HUD Layout Editor.
*   **F7**: Toggle Minimap.
*   **F8**: Toggle FPS/Coords.

## Building
**GitHub Actions**: Push to GitHub, and the Actions tab will build it automatically (Gradle 9.2).
